# ALUMCO — Plataforma de Capacitación Interna

Plataforma web de capacitación interna para ONG ALUMCO.
Stack: Next.js 16, Supabase, Tailwind CSS, shadcn/ui, Vercel.

## Instalación local

1. Clonar el repositorio
2. `npm install`
3. Copiar `.env.example` como `.env.local` y completar las variables
4. `npm run dev`
