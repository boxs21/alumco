# app/profesor/capacitaciones/

Gestión de capacitaciones para el profesor. Misma funcionalidad que `/admin/capacitaciones`.

| Subruta | Descripción |
|---------|-------------|
| `/[id]` | Detalle y edición de capacitación (materiales, quiz) |
| `/[id]/asignar` | Asignar capacitación a colaboradores |
| `/nueva` | Crear nueva capacitación |
