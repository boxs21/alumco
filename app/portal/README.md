# app/portal/

Panel del colaborador. Accesible solo por usuarios con rol **COLLABORATOR**.

| Sección | Ruta | Descripción |
|---------|------|-------------|
| Inicio | `/portal` | Dashboard con capacitaciones asignadas y su estado de progreso |
| Capacitación | `/portal/capacitacion/[id]` | Detalle de una capacitación: materiales, quiz y certificado |
| Historial | `/portal/historial` | Historial completo de capacitaciones completadas |
| Calendario | `/portal/calendario` | Sesiones programadas asignadas al colaborador |
