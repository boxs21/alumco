# app/

Directorio raíz de Next.js App Router. Contiene todos los paneles de la plataforma ALUMCO organizados por rol.

| Carpeta | Rol | Descripción |
|---------|-----|-------------|
| `(auth)/` | Todos | Login y recuperación de contraseña |
| `admin/` | Admin / Profesor | Panel de gestión de capacitaciones y personal |
| `profesor/` | Profesor | Panel de gestión de capacitaciones propias |
| `portal/` | Colaborador | Portal personal para completar capacitaciones |
| `api/` | — | Rutas de API REST internas |
