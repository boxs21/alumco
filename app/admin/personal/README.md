# app/admin/personal/

Gestión de personal. **Solo accesible por ADMIN** (bloqueado para PROFESOR en middleware).

**Funcionalidades:**
- Ver todos los usuarios del sistema (todos los roles)
- Crear nuevos usuarios (dispara API `/api/admin/create-user`)
- Editar rol, sede, área y estado de cualquier usuario
- Activar / desactivar cuentas
