# app/admin/capacitaciones/[id]/

Página de detalle y edición de una capacitación específica.

**Permite:**
- Editar nombre, descripción, área, sede y estado (BORRADOR / PUBLICADO / ARCHIVADO)
- Subir y ordenar archivos adjuntos (PDF, video, presentación, imagen) o embeds de YouTube/Drive
- Crear y editar el quiz asociado (preguntas, alternativas, puntaje mínimo, intentos máximos)
- Ver estadísticas de asignación y completado
