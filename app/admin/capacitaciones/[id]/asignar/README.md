# app/admin/capacitaciones/[id]/asignar/

Página para asignar una capacitación a colaboradores.

**Opciones de asignación:**
- **Todos** — todos los colaboradores activos
- **Por sede** — Concepción o Coyhaique
- **Por área** — Cuidado, Enfermería, Administración, etc.
- **Individual** — selección manual de colaboradores

Evita duplicar asignaciones existentes. Permite definir fecha límite opcional.
