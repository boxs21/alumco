# app/admin/colaboradores/[id]/

Perfil detallado de un colaborador específico.

**Muestra:**
- Datos personales: nombre, email, área, sede, estado
- Historial de capacitaciones asignadas con estado (PENDIENTE, EN PROGRESO, COMPLETADO)
- Certificados obtenidos con enlace de descarga
- Estadísticas de cumplimiento individual
