# app/admin/dashboard/

Panel principal del administrador. Muestra estadísticas generales de la organización.

**Métricas:**
- Colaboradores activos
- Capacitaciones publicadas
- Porcentaje de cumplimiento general
- Certificados emitidos

**Secciones:**
- Comparativa de progreso por sede (Concepción / Coyhaique)
- Progreso por área (Cuidado, Enfermería, Administración)
- Feed de actividad reciente (últimas capacitaciones completadas)
